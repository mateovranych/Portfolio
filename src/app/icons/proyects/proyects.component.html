SCDENTAL

MERCADOCHAMBA

Me desempeñé como desarrollador backend en una simulación de la organización No Country. Utilicé diversas tecnologías para poder agilizar el poco tiempo que tuvimos de desarrollo. Trabajamos en equipo bajo la metodología SCRUM y tuvo una duración de 5 semanas.

Tecnologías utilizadas: .NET 6.0, Angular, SCRUM, Entity, Trello, Slack

OWNSTYLE

Es una ecommerce en la cual se pueden agregar productos, clientes y usuarios. Está hecha con Angular en el frontend y .NET en el backend. Para levantar las imágenes utilicé IIS, pero me gustaría implementar AWS y alguna metodología de pago automática como Mercado Pago.

Tecnologías utilizadas: .NET 6.0, Angular

MARAVI

Para el final de la tecnicatura, junto a mi grupo desarrollamos una aplicación web para una tienda de ropa. La aplicación la desarrollamos en el frontend con Angular, en el backend desarrollamos una API en .NET 6 y para la base de datos utilizamos MySQL. Además integramos la API de Mercado Pago para realizar los cobros.

Tecnologías utilizadas: .NET 6.0, Angular, Entity, AWS
