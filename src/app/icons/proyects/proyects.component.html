Maravi

Para el final de la tecnicatura, junto a mi grupo desarrollamos una aplicación web para una tienda de ropa. La aplicación la desarrollamos en el Front End con Angular, en el Back End desarrollamos una API en .NET 6 y para la base de datos utilizamos MySql. Además intregramos la API de Mercado Pago para realizar los cobros

OwnStyle

Es una ecommerce en la cual se pueden agregar productos, clientes y usuarios, está hecha con Angular en el frontend y .NET en el backend, para levantar las imagenes utilicé IIS pero me gustaría implementar AWS y alguna metodología de pago automática como Mercado Pago.

SCDENTAL
